# Dynamic Rust

A comprehensive guide to modular programming in Rust.

## Overview

- Keybinding
  - With configuration files.
  - With a scripting engine.
- Plugins
  - With dynamic libraries.
  - With a scripting engine.
- Event Hooks
  - With a scripting engine.
  - With configuration files.
  - With services.
